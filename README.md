# 🎯 Number Guessing Game

An interactive GUI-based Number Guessing Game developed using **Python** and **Tkinter**. The game generates a random number between 1 and 100, and the player must guess it using the hints provided. A scoring system rewards players who find the correct number in fewer attempts.

##  Features

-  Random number generation between 1 and 100
-  Clean and user-friendly Tkinter GUI
-  Real-time hints ("Too High" / "Too Low")
-  Attempt counter to track guesses
-  Dynamic scoring system based on performance
-  Success message when the correct number is guessed
-  Restart/New Game functionality
-  Instant feedback for every guess

## 🏆 Scoring System

| Attempts | Score |
|-----------|--------|
| 1 - 3     | 100    |
| 4 - 5     | 80     |
| 6 - 8     | 60     |
| 9 - 12    | 40     |
| 13+       | 20     |

The fewer attempts you take, the higher your score!

##  Technologies Used

- Python
- Tkinter
- Random Module
